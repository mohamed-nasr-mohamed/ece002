visual programming language

In computing , the language of visual programming ( in English : the Visual Research Programs are newer ) is any programming language that allows users to create programs by manipulating the elements of a program graphically rather than defined literally, ie , it allows programming by visual expressions and arrangements for spatial texts, diagrams and symbols. For example, many visual programming languages ​​known as dataflow or graphic programming are based on the idea of ​​"boxes and arrows", where boxes or other screen objects are treated as entities connected by arrows, lines, or parentheses which represent relationships.
